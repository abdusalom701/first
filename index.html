CASCADING Style SHeets CSS

CSS veb-sahifa yaratuvchilari tomonidan ranglar, shriftlar, alohida bloklarning tartibini va ushbu veb-sahifalarni paydo bo'lishining boshqa jihatlarni ishlatish uchun kerek bo'ladi. CSS -ning asosiy maqsadi veb-sahifaning mantiqiy tuzilishining tavsifini ushbu veb-sahifa tashqi korinishi tavsifidan ajratish edi. Ushbu ajratish hujjatning mavjudligini oshirishinin, koproq moslashuvchanligi va taqdimotni bajarish qobiliyatini taminlab,shuningdek tarkibiy tarkibdagi murakkablik va takrorlanuvchanlikni kamaytirish mumkin.
